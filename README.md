# Breast Cancer Classification (ML)

## Overview

Machine learning-based classification of breast tumors (malignant vs benign) using the Wisconsin Diagnostic Breast Cancer dataset.

## Problem

Early detection of breast cancer is critical. This project builds models to classify tumors based on cell features extracted from medical data.

## Dataset

* Wisconsin Diagnostic Breast Cancer (WDBC)
* 569 samples with 30 numerical features
* Source: UCI Machine Learning Repository

## Method

* Data preprocessing and feature analysis
* Model training using machine learning algorithms (e.g., Logistic Regression, Random Forest)
* Evaluation using accuracy and classification metrics

## Results

Model performance is evaluated using accuracy and confusion matrix (see notebook for details).

## How to Run

```bash
jupyter notebook breast_cancer_classification.ipynb
```
